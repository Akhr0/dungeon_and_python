# Parcours 20 etapes (Pygame debutant)

Chaque fichier est executable seul.
Chaque fichier se termine par un bloc **EXERCICE GUIDE** (pas a pas).
Le suivi des notions (premiere apparition) est dans `cours_20_etapes/JOURNAL_NOTIONS.md`.

Ordre pedagogique respecte au debut:
0. Variables
1. Boucles
2. Conditions
3. Listes
4. Fonctions
5. Dictionnaires

Puis on ajoute des surcouches incrementales jusqu'au jeu final.

Principe incremental:
- chaque etape garde la base de la precedente;
- on ajoute une seule grande nouveaute a la fois;
- certains elements sont marques "temporaires" puis remplaces plus tard.

## Lancement

Depuis la racine du projet:

```bash
source .venv/bin/activate
python cours_20_etapes/step_00_variables.py
```

Remplace le nom du fichier par l'etape voulue.
