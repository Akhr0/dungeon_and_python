# Fiches Eleves Imprimables (00 -> 20)

Ces fiches ne sont plus generiques: chaque fiche est reliee au contenu reel du step.
Elles contiennent:
- un aide-memoire de la notion
- un approfondissement (au dela du step)
- une checklist de verification eleve

## Utilisation conseillee en classe
1. Distribuer la fiche du step du jour.
2. Lancer le script correspondant.
3. Faire le bloc EXERCICE du code + checklist de la fiche.
4. Faire une restitution orale de 2 minutes par eleve.
