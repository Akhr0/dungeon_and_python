# Parcours Eleve - Version A Completer

Ce dossier contient les memes steps que `cours_20_etapes/`, mais avec des zones a remplir.

Balises utilisees dans les fichiers (tres visibles):
- `# >>> START CODE ELEVE`
- `# <<< END CODE ELEVE`

Objectif pedagogique:
- l eleve ne recopie pas tout le fichier;
- il tape seulement les lignes cle de la notion du step.

Conseil d usage en classe:
1. Ouvrir le step original sur ecran partage.
2. Ouvrir le meme step dans ce dossier cote eleve.
3. Faire ecrire uniquement les zones balisees.
4. Lancer le script une fois complete.
